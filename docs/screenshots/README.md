# 截图清单

请按下面清单提供截图，方便统一展示在仓库文档中。

## 文件与内容

- `dashboard-overview.png`：主页总览（顶部状态卡 + 标签页）
- `config-panel.png`：配置区（app_id/app_secret、token、保存）
- `drive-tree.png`：Drive 目录树（含刷新/展开/折叠操作区）
- `runtime-status.png`：飞书连接状态、自动同步倒计时、最近同步摘要
- `logs-panel.png`：日志巡检区（level/module 过滤）

## 建议规格

- 分辨率：`1920x1080` 或更高
- 格式：`PNG`
- 内容：使用真实 UI，不打马赛克关键控件（敏感 token 可遮挡）

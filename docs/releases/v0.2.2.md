# v0.2.2 Release Notes

## 发布信息

- 版本：`v0.2.2`
- 日期：`2026-02-22`
- 基线版本：`v0.2.1`

## Added

- 飞书事件回调触发同步：
  - `POST /api/events/feishu`
  - `GET /api/status/event-callback`
- 事件状态前端面板与配置项：
  - `sync.event_callback_enabled`
  - `sync.event_verify_token`
  - `sync.event_encrypt_key`
  - `sync.event_debounce_sec`
  - `sync.event_trigger_types`
- 新增事件回调测试：`app/tests/test_event_callback.py`。

## Changed

- 同步执行模型升级为“轮询兜底 + 回调加速”。
- CLI 与文档补齐事件回调配置、部署说明和排障路径。

## Fixed

- 增强高频事件下的去抖、去重、并发保护，降低重复触发带来的冲突风险。

## 升级说明

- 数据库迁移：无强制迁移。
- 配置兼容性：默认回调关闭；启用时需配置 `event_verify_token`。
- 需要人工操作：在飞书开放平台订阅事件并回填回调配置。

## 回滚说明

- 目标回滚版本：`v0.2.1`
- 回滚命令：`scripts/rollback_release.sh --target v0.2.1 --restart-service`

## 验证记录

- `make check`：通过
- `curl /api/healthz`：`ok=true`
- `curl /api/readyz`：`ok=true`
